tag code

tem uma tag muito legal pra quando quero escrever algo que precisa ter um monoespaçamento
ou seja, todos os caracteres com o mesmo espaçamento.
Com essa tag consigo escrever linhas de código de forma clara, como por exemplo:
setInteractive()
on.('pointerdown', => {
this.x = 190
})

caso eu não queira dar br a cada linha 

  setInteractive()
on.('pointerdown', => {
    this.x = 190
})

outra parada bem legal é quando eu abrevio algo e quero que apareça o significado ao passar o mouse por cima
Para isso eu coloca a tag abbr então quando eu escrevo vc é bnt
